  • Créer une réponse automatique

    Afin de créer un message d’absence ou une réponse automatique sur Microsoft Outlook, rendez-vous sur l’accueil de votre boite mail :


    https://outlook.office.com/mail/inbox


    Cliquez ensuite sur le logo des paramètres en haut à gauche

     

     

    Cliquez sur « Afficher tous les paramètres d’Outlook »

     

     

    Une fois le menu ouvert, vous serez redirigé vers ce lien :
    https://outlook.office.com/mail/options/mail/layout

     

     

    Cliquez sur « Réponses automatiques » puis activer celles-ci.

     

     

    À partir de ce menu, vous pouvez désormais définir votre réponse automatique notamment lors de vos absences.
    Vous pouvez également définir une période durant laquelle les réponses automatiques fonctionneront (en cas de congés par exemple).

     

     

    ⚠️ N’oubliez pas d’enregistrer avant de fermer la fenêtre.

     

  • Activer le transfert de sa boîte mail

    Afin de créer un message d’absence ou une réponse automatique sur Microsoft Outlook, rendez-vous sur l’accueil de votre boite mail :


    https://outlook.office.com/mail/inbox


    Cliquez ensuite sur le logo des paramètres en haut à gauche

     

     

    Cliquez sur « Afficher tous les paramètres d’Outlook »

     

     

    Une fois le menu ouvert, vous serez redirigé vers ce lien :
    https://outlook.office.com/mail/options/mail/layout

     

     

    Cliquez sur le bouton "Transfert"

     

     

    Cochez ensuite la case "Activer le transfert", puis renseigner l'adresse mail du destinataire

     

     

    N'oubliez pas d'enregistrer, le transfert devrait être effectif.

  • Créer une règle

    Les règles vous permettent de déplacer, marquer et répondre aux messages électroniques automatiquement. Vous pouvez également utiliser des règles pour lire des sons, déplacer les messages vers des dossiers ou afficher les alertes de nouveaux éléments.

    La règle la plus simple et la plus courante à créer est celle qui vous permet de déplacer un élément d’un certain expéditeur ou avec certains mots dans la ligne d’objet vers un autre dossier.

     

    Voici la méthode à suite afin de créer une règle.

     

    Rendez-vous sur votre boîte mail Outlook via ce lien:

     

    https://outlook.office.com/mail/inbox

     

    Cliquez ensuite sur "Paramètres" en haut à droite.

     

     

    Un onglet s'ouvre, cliquez alors sur "Afficher tous les paramètres d'Outlook."

     

     

    Vous arriverez sur ce nouvel onglet:

     

     

    Cliquez sur "Règles"

     

     

    Nous allons ajouter une nouvelle règle avec comme exemple le transfert de mail provenant de kevin.ledru@abcdnutrition.com vers remi.dandelot@abcdnutrition.com

     

    J'applique ma règle, celle-ci devrait désormais fonctionner.

     

    Beaucoup de conditions et d'actions sont disponibles, à vous de définir celle qui convient pour votre règle.

     

    N'oubliez pas d'enregistrer avant de fermer la fenêtre afin d'appliquer votre règle !

  • Créer votre signature

    Afin de créer votre signature sur Outlook version Logiciel, cliquez sur "Nouveau Courrier" dans l'application

     

     

    Un nouvel onglet de rédaction s'ouvre, cliquez alors sur "Une signature" puis sur "Signatures"

     

     

    Vous pouvez alors créer une nouvelle signature

     

     

    Il vous est possible de créer plusieurs signatures et de sélectionner la bonne lors de l'envoi d'un mail.

     

    N'oubliez pas d'activer votre signature pour chaque nouveau message et chaque réponse

     

     

    Cliquez sur "Ok", votre signature est désormais fonctionnelle.
